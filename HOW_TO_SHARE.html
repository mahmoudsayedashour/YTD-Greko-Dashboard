🚀 Greko Egypt Dashboard – Sharing Guide

Option 1: Share as a ZIP file

Zip the entire YTD Greko Dashboard folder and send it. The recipient just double-clicks index.html.

Option 2: Host on GitHub Pages (Free, shareable link)

  1. Create a free account at github.com
  2. Create a new public repository
  3. Upload all files from the YTD Greko Dashboard folder
  4. Go to Settings → Pages → Deploy from main branch
  5. You get a free link like: https://yourusername.github.io/greko-dashboard/

Option 3: Netlify Drop (Easiest – 30 seconds)

  1. Go to app.netlify.com/drop
  2. Drag the entire YTD Greko Dashboard folder onto the page
  3. You instantly get a shareable link!
